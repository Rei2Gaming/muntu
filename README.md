# muntu

Muntu Version 0.1 Test. Be kind, be honest, be brutal.

A few things before you start:

This is an early prototype — rough edges are expected. I'm testing whether the experience works, not whether the code is clean.

While you play, say whatever's on your mind out loud. If something confuses you, say it. If something feels off, say it. If nothing happens — that's useful too.

Play all the way through once. It takes about 10 minutes. When you finish, fill in a short form — the link is below. Don't open the form until you've finished playing.

Thank you so much again for your time!
